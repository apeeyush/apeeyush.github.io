The Concord Consortium GSoC Proposal

Data analytic for user actions in HTML5 web apps

This project will involve creating a common system for sending user action logging data from HTML5 activities into visualization application, CODAP for analytics by researchers. This will also involve creating a system to allow non-developers to apply transformations to the data before sending it to CODAP and modifying CODAP to be able to pull data from the database. An additional description can be found on The Concord Consortium's ideas page.

The latest version of proposal can be found here.

A demo app built as a proof of concept is available here. The code for this web app is present on Github.
